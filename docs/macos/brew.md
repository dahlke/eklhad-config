# Homebrew

Use the automated installer:

```bash
./scripts/bash/macos/setup.sh
```

This script installs Homebrew (if needed), applies the `Brewfile`, runs
`apply-configs.sh`, and cleans up.

To export your current machine into `Brewfile`:

```bash
./scripts/bash/export-brewfile.sh
```

## Install Commonly Used Tools and Apps (manual opens)

```bash
./scripts/bash/macos/open-apps.sh
```
